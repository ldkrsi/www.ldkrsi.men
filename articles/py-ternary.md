# python的三元運算子(ternary conditional operator)

三元運算子(ternary conditional operator)

在C/C++裡以 ?: (問號冒號)表示

在python中則以`x if y else z`表示

```python
x = True if 'a' == 'a' else False
 
#意思同於
if 'a' == 'a':
    x = True
else:
    x = False
```
